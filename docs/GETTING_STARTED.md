# Getting Started with BPO Address Capture System

## Installation

```bash
npm install
```

## Running the Application

```bash
npm start
```

Server will start on http://localhost:3000

## Project Structure

```
bpo-address-capture/
├── server.js              # Express server
├── public/                # Frontend files
├── models/                # Database models
├── routes/                # API routes
├── middleware/            # Express middleware
├── utils/                 # Utility functions
└── docs/                  # Documentation
```

## Features

- 📞 Live call management
- 📍 Address capture and extraction
- ✅ QA workflow
- 📊 Analytics dashboard
- 🔐 Secure authentication
