# 🏢 BPO Address Capture System

Professional-grade address capture system for BPO call centers with live call integration.

## ✨ Features

- 📞 Live call management
- 📍 Multi-method address capture (voice/text/recording)
- 🧠 Intelligent address extraction
- ✅ QA workflow system
- 👥 Agent monitoring
- 📊 Real-time analytics
- 🔐 Enterprise security

## 🚀 Quick Start

```bash
npm install
npm start
```

Access at http://localhost:3000
